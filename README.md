# jianbihua
简笔画 爬虫 图像 美术 数据集 动物简笔画


网址：https://www.jbhdq.com
最近因为要用GAN来做实验，需要用到简笔画，所以我在网上爬了一下动物简笔画，当然，这个代码可以很容易扩展到整个网站的简笔画
本代码使用了多线程，高效，快速
没有用异步，因为没有必要
